## 第一天记录  7.15
----
### 第一部分：过程记录
----
**1、预配置**

 在第一天的开始，我拿到了一块空的u盘和镜像源，在ventory工具的帮助下，我成功将镜像源拷贝到u盘里面并着手下一步

----
**2、整机安装**

 随后我开始安装整机，将u盘插入usb口，进入BIOS界面，调整启动优先级并按f10保存设置，进入安装界面，选择安装的硬盘位置，配置文件，用户名等。

----
**3、重校镜像源，安装ubuntu**

重新选择了Ubuntu的镜像源盘，下载安装成功，配置用户，并检查了系统的更新，连接到网络，使用sudo apt install 软件包名 -y安装ssh，pip等软件包，确认IP地址，为下一步做准备.

----
**4、远程连接Windows，为后续办公做准备**

在Ubuntu的配置环节，已经确认好了ssh和ip地址，转到Windows机上，打开命令行界面，输入ssd和用户名连接，然后输入密码，完成了Windows和Linux的远程连接

----
**5、GitHub互通以及git基础**

在引导下，我初步学习了如何在有授权的情况下进入仓库合作开发，我成功完成了在ssh连接上克隆仓库，打开，运行文件
